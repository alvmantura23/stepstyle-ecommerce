# StepStyle - E-Commerce de Calzado 🚀
StepStyle es una aplicación de e-commerce moderna construida con React JS y Tailwind CSS, especializada en la venta de calzado.

## Características
🛍️ Catálogo de 16 productos con imágenes y precios
🔍 Filtrado por precio, novedades y ofertas
🛒 Carrito persistente en localStorage
💖 Lista de deseos personal
📱 Diseño 100% responsive
🎨 Interfaz con animaciones suaves
🗺️ Integración con Google Maps API
📝 Sistema de blog integrado

## 🚀 Instalación

1. Clonar repositorio:
git clone https://github.com/tuusuario/stepstyle.git
cd stepstyle

2. Instalar dependencias:
npm install

3. Iniciar servidor de desarrollo:
npm run dev

4. Construir para producción:
npm run build
